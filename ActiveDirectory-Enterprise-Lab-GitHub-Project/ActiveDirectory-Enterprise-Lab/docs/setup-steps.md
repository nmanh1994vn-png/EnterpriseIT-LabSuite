# Setup Steps

1. Installed VMware Workstation and created two virtual machines.
2. Installed Windows Server 2022 on the first VM.
3. Configured static IP address for the Domain Controller.
4. Installed Active Directory Domain Services.
5. Promoted the server to a Domain Controller.
6. Created Organizational Units (USA, Asia, Europe).
7. Created domain users and security groups.
8. Installed Group Policy Management Console.
9. Created and tested Group Policy Objects.
10. Installed Windows 10 on a second VM.
11. Joined the Windows 10 machine to the domain.
12. Configured file sharing and mapped network drives.
