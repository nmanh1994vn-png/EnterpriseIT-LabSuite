Active Directory Lab Setup Steps

1. Installed VMware Workstation and created virtual machines.
2. Installed Windows Server 2022 on the domain controller VM.
3. Configured static IP and DNS settings.
4. Installed Active Directory Domain Services (AD DS).
5. Promoted the server to Domain Controller.
6. Created Organizational Units: USA, Asia, Europe.
7. Created domain users and security groups.
8. Installed Group Policy Management Console.
9. Created and tested Group Policy Objects.
10. Installed Windows 10 client VM.
11. Joined client machine to the domain.
12. Tested login with domain user accounts.
13. Configured file sharing and mapped network drive.
14. Verified Group Policy application using gpupdate /force.
