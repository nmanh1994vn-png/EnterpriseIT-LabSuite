# Microsoft 365 / Entra ID / Intune Lab

## Lab Objective

The objective of this lab is to simulate enterprise cloud identity and device management using Microsoft 365 services including Entra ID and Intune.

---

## Technologies Used

- Microsoft 365
- Entra ID (Azure Active Directory)
- Microsoft Intune
- Windows 11 Client

---

## Tasks Performed

1. Created Microsoft 365 tenant
2. Created Entra ID users and groups
3. Assigned Microsoft 365 licenses
4. Configured device enrollment in Intune
5. Applied device compliance policies
6. Managed devices using Microsoft Intune

---

## Key Skills Demonstrated

- Cloud identity management
- Microsoft 365 administration
- Device management with Intune
- User and group administration
- Enterprise cloud security concepts
