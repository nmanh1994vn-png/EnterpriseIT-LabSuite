# Microsoft 365 / Entra ID / Intune Lab

## Overview

This lab demonstrates cloud identity and device management using **Microsoft 365**, **Entra ID**, and **Microsoft Intune**.

The environment simulates a basic enterprise cloud setup where users, groups, and devices are managed through Microsoft's cloud identity platform.

Key components configured in this lab include:

• Microsoft 365 tenant setup  
• Entra ID user and group management  
• Microsoft Intune device enrollment  
• Device compliance policies  
• Cloud identity authentication

---

## Lab Architecture

```
User Device
│
└── Windows 11 Client
    │
    └── Microsoft 365 Tenant
        │
        ├── Entra ID
        │   ├── User Management
        │   └── Group Management
        │
        └── Microsoft Intune
            ├── Device Enrollment
            ├── Device Compliance
            └── Policy Management
```

## Lab Objectives

The goal of this lab is to demonstrate practical experience with:

• Cloud identity management  
• User and group administration in Entra ID  
• Device enrollment with Microsoft Intune  
• Cloud device compliance policies  
• Enterprise cloud administration
