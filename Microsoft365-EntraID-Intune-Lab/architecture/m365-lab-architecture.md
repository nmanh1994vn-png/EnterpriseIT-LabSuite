# Microsoft 365 / Entra ID / Intune Lab Architecture

## Overview

This lab demonstrates Microsoft 365 cloud identity and device management using Entra ID and Intune.

The environment simulates a typical enterprise cloud setup where users, groups, and devices are managed centrally using Microsoft cloud services.
